# 🌤️ Weather Dashboard

A full‑stack weather application built with **React (frontend)** and **Express (backend)**, deployed on **Render**.  
Users can search for any city and view real‑time weather data including temperature, humidity, wind speed, and pressure.

---

## 🛠️ Tech Stack
- **Frontend:** React, CSS
- **Backend:** Node.js, Express, Axios
- **Deployment:** Render (Static Site + Web Service)
- **API Provider:** OpenWeatherMap

---

## ⚙️ Features
- 🔍 Search weather by city name
- 🌡️ Displays temperature, condition, humidity, wind speed, and pressure
- 🖼️ Weather icons from OpenWeatherMap
- 📱 Responsive design for desktop and mobile
